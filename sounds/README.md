# Sounds Directory

Place your audio files here for the calming tools feature.

## File Naming Convention

Name your audio files exactly as follows:
- `rain.mp3` or `rain.wav` - For rain sounds
- `ocean.mp3` or `ocean.wav` - For ocean wave sounds
- `forest.mp3` or `forest.wav` - For forest/nature sounds
- `piano.mp3` or `piano.wav` - For piano/musical sounds

## Supported Formats

- MP3 (recommended - smaller file size)
- WAV (better quality, larger file size)
- OGG (if supported by browser)

## Where to Get Free Sounds

### Free Sound Websites:
1. **Freesound.org** - https://freesound.org
   - Search for: "rain", "ocean waves", "forest", "piano"
   - Filter by: License (CC0 or CC BY)
   - Download and rename to match the naming convention

2. **Zapsplat** - https://www.zapsplat.com
   - Free account required
   - High quality ambient sounds

3. **Pixabay** - https://pixabay.com/music/
   - Free music and sound effects
   - No attribution required

4. **YouTube Audio Library** - https://studio.youtube.com/channel/UC.../music
   - Free music and sound effects
   - No copyright issues

## Tips

- Keep file sizes reasonable (1-5 MB per file is ideal)
- Use MP3 format for better compatibility and smaller sizes
- Ensure files are loopable (seamless loop) for continuous playback
- Test files in browser before using

## Example Download Commands

If you have audio files, simply copy them to this directory:
```bash
cp /path/to/your/rain.mp3 /Applications/XAMPP/xamppfiles/htdocs/Mood-Diary-main/sounds/rain.mp3
```

## Note

If audio files are not found, the app will automatically use generated sounds (Web Audio API) as a fallback.
